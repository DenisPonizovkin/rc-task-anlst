# Данные

В файле data.csv хранится информация о числе объектов разного типа,
подлежащих обновлению и стоимостях различных типов инвестиционных
мероприятий по обновлению объектов.

В приведенных данных тип объекта соответствует полю
```object_tp_id```, тип мероприятия - полю ```arrangement_tp_id```.

Информация по числу объектов и величине инвестиций по их обновлению  обладает
однотипной структурой, поэтому хранится в единой таблице, но различается
по типу, зафиксированному в поле  ```te_form_tp_id```. Значение этого поля равное:
1324 - определеяет величину, соответствующую числу объектов,
подлежащих обновлению (единицы измерения - штуки)
1326 - определяет величину, соответствующую стоимостям инвест мероприятий

Эти два среза данных пересекаются по полям:
- ```object_tp_id``` - тип объекта обновления
- ```arrangement_tp_id``` - тип мероприятия обновления объекта

Приведенные данные являются изначальной справочной информацией.
При этом значения колонки val для 
```te_form_tp_id``` = 1326 являются неизменной величиной.
То есть число объектов является переменной варьируеморй величиной при решении задачи. 

Обозначим как $A$ срез данных, соответствующий ```te_form_tp_id``` = 1324.
Обозначим как $B$ срез данных, соответствующий ```te_form_tp_id``` = 1326.
$A \times B$ - пересечение (inner join) по полям ```object_tp_id``` и ```arrangement_tp_id```.

# Задача

Введем функцию $I(A, B)=sum(А'.val \cdot B'.val)$, $A', B' \in A \times B$ - вещественное число,
равное сумме произведений инвестиционных затрат на соответствующее число объектов
по типу объектов и мероприятий
(пересечение (inner join) по полям ```object_tp_id``` и ```arrangement_tp_id```).

$M=I(A, B)$ от изначальных $A, B$ является максимумом.
Пользователь уменьшает на некоторый процент величину $M$
(параметр функции) и получаем новую величину $M_1$ в результате уменьшения.

По задаче вариьируемый показатель - число объектов
(колонка val ```te_form_tp_id``` = 1324). Число объектов можно только уменьшать.
При корректировки изначальной таблицы $B$ путем уменьшения val получаем новый срез данных, который обозначим $B_1$.

По задаче необходимо подобрать такие уменьшения
величины val для разных типов объектов и мероприятий так, чтобы
$M_1 - I(A, B_1) \rightarrow \min$ и $\sum_{val}(B_1) \rightarrow \max$.
То есть необходимо максимально уменьшить число объектов, но так, чтобы
сумма инвестиционных мероприятий при этом изменилась минимально.
